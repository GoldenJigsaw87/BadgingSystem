# BadgingSystem
A modular badge system that uses devices in a qrcode, phoness allow users to create and use their badge, and the admin website allows us to monitor our devices and leaderboards


# Tech Stack
Frontend:
- iOS/Android - Expo (React Native)
- TabletView - Swift/SwiftUI
- Admin - Next.JS
Backend - Nest.js
Database: MongoDB (Firebase)
Authentication: Clerk
Hosting: Vercel
